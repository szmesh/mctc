# mctc

> 星河文旅云平台

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# build to local
npm run buildlocal

# build to remote
npm run buildremote

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test

# install depend
npm i element-ui -S
```
