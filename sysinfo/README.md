### nginx
##### 安装路径
> /etc/nginx
##### 默认服务目录
> /usr/share/nginx/html
##### 启动
```ssh
nginx
```
##### 重启
```ssh
nginx -s reload
```
##### 停止
```ssh
nginx -s stop
```
